Ashish Dodiya Portfolio API

Welcome to the backend API for Ashish Dodiya's portfolio website. This API handles contact form submissions and other backend functionality.

API Endpoints

POST /api/contact

Sends a message from the contact form to the site owner via email.

Request Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "subject": "Project Inquiry",
  "message": "I'm interested in working with you on a project."
}

Response (Success):

{
  "success": true,
  "message": "Your message has been sent successfully!"
}

Response (Error):

{
  "success": false,
  "error": "Error message details"
}

Implementation Notes

  • This API uses Express.js for routing and middleware.
  • Email delivery is handled through Nodemailer with Gmail SMTP.
  • CORS is enabled for the frontend origin.
